Auctions

Language: C#

2018-Present

This is a simple auction website for making and bidding on items. Users are granted fictional starting money, can bid on other items and earn their own via items they sell. Deployed with AWS here.

Technologies Used

  • ASP.NET CORE 2.1
  • Razor
  • Bootstrap 4.1.3
  • AWS Ubuntu deployment
  • Entity
  • MySQL 8.0
  • PostGres
